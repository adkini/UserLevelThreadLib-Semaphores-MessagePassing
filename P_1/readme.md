#Non Premptive OS User level scheduler


1) All files have to be in the same folder for getting compiled properly.
2) q.h has all the queuing routines. TCB.h has the TCB structure defined and init function defined. thread_test.c has the code for starting 	and yielding the threads.'Threads.h' has the run,yield,and start thread functions defined as instructed.
3) compile and run the thread_test.c program. The program runs infinetely yielding at a condition.
